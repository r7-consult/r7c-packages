Как использовать

Скопируйте папку hello-world, переименуйте и адаптируйте файлы. После сборки .plugin проверяйте его напрямую в {r7}/ONLYOFFICE (Plugins → Upload).

Полный гайд

5 шагов добавления

  1. 1

    Структура

    Создайте sdkjs-plugins/content/<plugin>/, добавьте config.json, index.html, scripts, resources.

  2. 2

    Иконки и скрины

    Подготовьте light/dark иконки и минимум один скриншот в resources/store.

  3. 3

    Документация

    Добавьте README.md, CHANGELOG.md, LICENSE, 3rd-Party.txt.

  4. 4

    Регистрация

    Добавьте плагин в store/config.json и проверьте категории.

  5. 5

    Проверка deploy

    Соберите deploy/<plugin>.plugin и загрузите его в {r7}/ONLYOFFICE для ручной проверки.

Чеклист перед PR

  • config.json валиден

    name/guid/version/variations заполнены, entry-point существует.

  • Иконки и скриншоты готовы

    Пути к light/dark и store/screenshots корректны.

  • store/config.json обновлён

    Плагин добавлен, категории соответствуют допустимым.

  • deploy собран

    Файл <plugin>.plugin находится в deploy/.

  • Документация приложена

    README + CHANGELOG + LICENSE + 3rd-Party есть в корне.
