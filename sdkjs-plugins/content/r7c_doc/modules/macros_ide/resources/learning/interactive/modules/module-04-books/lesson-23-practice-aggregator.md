# Урок 23. Практика Инженерии: Сборщик реестров (Aggregator)

Мы начинаем реализацию концепций из Урока 22. Написание скриптов-пауков, консолидирующих фрагментированные данные в единый реестр, — одна из самых частых задач промышленной автоматизации.

## 🎯 Бизнес-задача

Книга Р7-Офис состоит из 10-х вкладок ("Филиал-Север", "Филиал-Юг" и т.д.). В каждом листе на ячейке `B2` лежит Итог месяца. Ваша архитектурная задача: стянуть все значения в единый реестр.

В среде VBA эта процедура осуществлялась через `For Each ws In Worksheets : ws.Range("B2").Value` и т.д. В Р7-Офис мы используем перебор массива коллекций листов.

Макрос должен:
1. Исключить целевой (сводный) отчет из проверки с помощью строковых операторов неравенства.
2. Провести итерацию по каждой вкладке-филиалу.
3. Прочитать скалярное значение ячейки `B2`.
4. Считать системное `Имя Вкладки`.
5. Записать полученные переменные на лист `Сводная` в колонки A и B.
6. Выполнить инкремент курсора (`writeRow++`), чтобы избежать перезаписи (Overwrite) при переходе к следующему филиалу.

---

## 📝 Разбор Алгоритма (Variables Scope)

В стартовом коде переменная (счетчик-якорь) `writeRow` определена **вне** блока `for`.

```javascript
let writeRow = 2; // Инициализация начала координат (Глобально для цикла)

for (let i = 0; i < allSheets.length; i++) {
    // Внутренняя лексическая область (Local Scope)
}
```

Все переменные (`let`), определенные внутри `{ ... }` цикла, уничтожаются и пересоздаются каждую итерацию.
Переменные (Якоря, Накопители), которые объявлены до цикла, живут на протяжении всей функции (IIFE). Именно поэтому агрегатор помнит актуальную строку записи между итерациями, и магазины не затирают друг друга.

## ✍️ Решение (Справочник разработчика)

Для реализации пайплайна Сборки используйте следующий шаблон:

```javascript
// Фильтр от петель парсинга самого себя
if (sName !== "Сводная") {
    
    // Сбор метрик с листов (Extract)
    let revenue = currentSheet.GetRange("B2").GetValue();
    
    // Пакетная выгрузка ячеек в реестр (Load)
    summarySheet.GetRange("A" + writeRow).SetValue(sName);
    summarySheet.GetRange("B" + writeRow).SetValue(revenue);
    
    // Сдвиг целевого указателя
    writeRow++; 
}
```

*Обратите внимание на динамическое формирование строкового адреса (Конкатенация): `"A" + writeRow` конвертируется в `"A2"`, `"A3"`, `"A4"` и т.д.*
