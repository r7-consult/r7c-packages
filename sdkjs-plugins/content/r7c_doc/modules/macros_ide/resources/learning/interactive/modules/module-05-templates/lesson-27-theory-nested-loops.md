# Урок 27. Анализ Эталона: Шаблон Р7 «Чередовать цвета строк»

Откройте стандартный Шаблон Р7-Офис **"Чередовать цвета строк" (Alternate row colors)**. Вы увидите, что разработчики платформы не используют там знакомый нам `sheet.GetRange("A1:C10")`. Они используют **другие методы API**, раскрывающие объектно-ориентированную природу книги. Мы разберем их работу и воспроизведем скрипт в редакторе!

## 🎯 Разбор Уникальных Методов (API Discovery)

### 1. Как понять "Размеры таблицы"? (`GetUsedRange()`)
Если у вас 10 000 строк и 50 колонок, вам не нужно вручную вычислять `"A1:XFD10000"`.
В шаблоне Р7 используется метод **`.GetUsedRange()`**. Он сам вычисляет прямоугольник (Bounding Box), в котором есть хотя бы одна заполненная ячейка, и возвращает его в виде объекта Range! Это супер-мощный инструмент для "выделения всей таблицы разом".

### 2. Коллекция Строк (`GetRows()` и `GetRow(index)`)
Вместо собирания адресов вида `"A" + i`, разработчики шаблона получают коллекцию рядов!
```javascript
let myTable = sheet.GetUsedRange();
let allRows = myTable.GetRows(); // Получаем коллекцию (List)
let maxLines = allRows.GetCount(); // Спрашиваем, сколько их?
```

Внутри цикла они берут физическую строчку целиком через `.GetRow(int индекс_с_нуля)`. 
Это совершенно другой подход к адресации, более безопасный и быстрый.

## 📝 Постановка Задачи

1. Справа уже инстанцированы объекты `GetUsedRange()` и `totalRowsCount`. Ваша задача проста.
2. Взгляните на инициализацию цикла: `for (let i = 0; i < totalRowsCount; i += 2)`
3. Вместо `i++` (прибавлять 1), шаблон Р7 использует `i += 2`. Это означает прыжки через одну строку (0, 2, 4, 6)!
4. Внутри цикла уже определена переменная `targetRow`, считывающая конкретный Ряд из Bounding Box.
5. Напишите код: **Окрасьте `targetRow` цветом `yellow`**. 

## 🧠 Глубинка API (Аналог Rows(i) в VBA)
В Visual Basic for Applications вы писали:
`ActiveSheet.UsedRange.Rows(i).Interior.Color = vbYellow`

В API Р7-Офис шаблон показывает элегантную инкапсуляцию:
`sheet.GetUsedRange().GetRow(i).SetFillColor(yellow);`

*В этом вся суть напутствия: Шаблоны учат вас, как правильно взаимодействовать с движком V8 редактора Р7. Вы учитесь читать классы и методы (GetRows, GetCount), которые лежат на поверхности.*
