# Урок 32. Боевая Симуляция: Data-Аудит и Mock-Данные

В этом разделе мы переходим к **Гибридным симуляциям**. Больше никаких пустых листов! Мы имитируем реальную среду Р7-Офис.

## 🛠 Механика "Песочницы" (Mock Data)
Когда вы нажмете кнопку **"Начать практику"** (или запустите код в первый раз), наш бэкенд вызовет невидимый скрипт инициализации `beforeScript`. Этот скрипт мгновенно "вбросит" в вашу таблицу заготовленную базу данных (Mock Data) — в данном случае, список менеджеров (A) и их выручку (B).

> **💡 Важно:** Если во время экспериментов ваш макрос "сломает" таблицу (например, вы случайно сотрете данные или закрасите не те ячейки) — просто нажмите кнопку **"Сбросить лист" (Reset)** в интерфейсе. Состояние таблицы мгновенно вернется в изначальные настройки с мок-данными, и вы сможете попробовать написать макрос заново!

## 🎯 Бизнес-задача

На листе сгенерирован отчет менеджеров за Декабрь. Оказалось, что некоторые менеджеры сработали "в минус" (отрицательная выручка). 

В редакторе права заготовлен защищенный макрос (шаблонный стандарт). **Вам необходимо:**
1. Написать цикл, который просканирует строки базы данных со 2-й по 6-ю.
2. Считать значение Выручки из ячейки `B` для каждой строки.
3. Организовать `if / else` логику:
   - Если число **меньше 0** — фон ячейки `B{i}` перекрашивается в красный через `SetFillColor(Api.CreateSolidFill(redColor))`.
   - Если число **больше 0** — оно суммируется в общий котел (переменная `sum`).
4. После завершения цикла выгрузите итоговую маржу (накопленный `sum`) в подготовленную целевую ячейку `D2`.

Как только ваш макрос выполнит работу, система автотестов просканирует Лист. Она проверит, закрашены ли "минусовые" ячейки B3 и B5, и правильная ли сумма 250 000 рублей лежит в ячейке D2.

Потрясающая возможность попрактиковаться без риска испортить данные! Напишите код и нажмите "Проверить"!

### Готовое решение

Если запутались с циклами и IF, вот решение всей задачи:

```javascript
(function() {
    'use strict';
    try {
        const api = Api;
        let sheet = api.GetActiveSheet();
        let redColor = api.CreateColorFromRGB(255, 0, 0);
        let sum = 0; 
        
        for(let i = 2; i <= 6; i++) {
            // Читаем значение из столбца B для текущей строки и приводим к числу
            let val = Number(sheet.GetRange("B" + i).GetValue());
            
            // Проверка условия
            if (val < 0) {
                // Красим фон ячейки с выручкой ("минус")
                sheet.GetRange("B" + i).SetFillColor(api.CreateSolidFill(redColor));
            } else if (val > 0) {
                // Приплюсовываем (сумма = сумма + значение)
                sum = sum + val;
            }
        }
        
        // После того как весь цикл завершился и мы просканировали все строки,
        // выгружаем накопившуюся сумму в D2
        sheet.GetRange("D2").SetValue(sum);
        
    } catch(error) {
        console.error(error.message);
    }
})();
```
