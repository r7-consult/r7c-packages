**Урок 2.3. Разработка HTML-интерфейса (index.html)**

Файл **index.html** определяет внешний вид плагина. Даже самый простой интерфейс должен содержать базовую структуру HTML и подключение JavaScript-скрипта.

Пример минимального index.html:

**<!DOCTYPE html>**

**<html lang="ru">**

**<head>**

**<meta charset="UTF-8" />**

**<script src="code.js"></script>**

**</head>**

**<body>**

**<button id="btn-1">Сменить цвет</button>**

**</body>**

**</html>**

**Рекомендации:**

-    Используйте тег <script> в разделе <head> для подключения логики (code.js).
-    Можно подключать внешние библиотеки (например, plugins.js из OnlyOffice SDK), но это не обязательно для простых плагинов.
-    Элементы управления (кнопки, поля ввода) должны иметь уникальные id, чтобы к ним можно было обращаться из JavaScript.
-    Стилизация возможна через тег <style> или внешние CSS-файлы.

Интерфейс отображается во встроенном окне плагина справа от рабочей области редактора.
