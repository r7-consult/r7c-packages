# {r7c}.mlearn

`{r7c}.mlearn` — это учебный центр по макросам, автоматизации и API R7/ONLYOFFICE. Он помогает быстрее разобраться, с чего начать, и перейти от чтения к практике без перегруза лишними деталями.

## Что внутри

- понятные обучающие материалы для уверенного старта;
- примеры, которые помогают увидеть реальные сценарии применения;
- уроки и готовые заготовки для самостоятельной практики;
- удобная точка входа для тех, кто хочет изучать автоматизацию постепенно и по делу.

## Кому полезно

- тем, кто только знакомится с автоматизацией в R7/ONLYOFFICE;
- тем, кто хочет быстро найти понятный пример под свою задачу;
- командам, которым нужен единый учебный материал для входа в тему.

## Почему это удобно

Вместо долгого поиска по разрозненным материалам пользователь получает одно место, где можно быстро понять возможности платформы и перейти к следующему шагу.
