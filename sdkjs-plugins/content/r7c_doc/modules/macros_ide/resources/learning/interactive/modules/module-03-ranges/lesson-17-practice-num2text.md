# Урок 17. Алгоритмика (Number-To-Text Algorithm)

Инженеру часто приходится компенсировать отсутствие встроенных функций Excel, прописывая их на уровне движка. Классический пример из бухгалтерии России и стран СНГ — генерация "Суммы прописью" (SpellNumber) в Договорах и Счетах на оплату.

В VBA для этого создавались гигантские пользовательские функции (UDF). В Р7-Офис V8 вы можете писать чистые и сверхбыстрые алгоритмы трансформаций (ETL) используя JS Arithmetic (Math object).

## 🎯 Бизнес-задача: Синтез Текста (До 99 руб.)
Ваш скрипт читает Int-значение из `A1` (до 99), парсит его на разряды (десятки, единицы) и конкатенирует `String` значение, записывая его в `B1`.

Для оптимизации `If/Else` спагетти-кода вы загружаете языковые константы в **Data Maps (плоские массивы)** `ones` (Единицы), `tens` (Десятки) и `teens` (Исключения 10-19).

---

## 📝 Разбор Алгоритма (Zero-based Index Arrays)
JS-массивы индексируются с `0`.
`ones[0] = ""` (Пустота)
`ones[5] = "пять"`

**Если Number = 45:**
1. Разряд десятков: Деление `45 / 10` дает `4.5`. Встроенный С++ метод ядра `Math.floor()` отрезает дробную часть -> `4`.
   `tens[4] == "сорок"`.
2. Разряд единиц (Modulo Operator): Оператор `%` (Остаток от деления) - классический прием вычленения цифр. `45 % 10` возвращает остаток `5`.
   `ones[5] == "пять"`.
3. Конкатенация: `"сорок" + " " + "пять"`.

Если число ровно `40`: `tens[4] + " " + ones[0]` = `"сорок" + " " + ""` = `"сорок "`. Метод `.trim()` на 54 строке очистит trailing-пробелы.

## ✍️ Постановка Задачи

В редакторе реализован скелет макроса-транслятора:
1. Инициализированы словари массивов.
2. Прописано базовое ветвление `if/else`, обрабатывающее диапазоны `0` и `10-19`.
3. В нижнем блоке `else` доработайте парсинг из массивов:
   Соберите переменную `result`, конкатенируя значения из `tens` (используя индекс `tenPart`) и `ones` (индекс `onePart`). Пробел `" "` используйте как разделитель.
4. В конце скрипта выгрузите (Write) `result` в интерфейсную ячейку `B1`.

```javascript
/* Строка 29 - String Builder Array Mapping */
result = tens[tenPart] + " " + ones[onePart];

/* Строка 38 - DOM Target Write */
sheet.GetRange("B1").SetValue(result);
```

Словари, конкатенация и `Modulo %` — это абсолютная основа генераторов текстов. Ознакомившись с нейвлегке, расширяйте скрипт с использованием `Math.floor(val/1000)` для миллионов и миллиардов.
