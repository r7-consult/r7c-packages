## Обзор

`data_processing` это каталог плагинов для обработки данных в редакторе таблиц R7/ONLYOFFICE.

Пакет объединяет:

- Сравнение файлов
- Анализ файлов
- ClickHouse
- OLAP
- Коммерческие карточки с переходом во внешние репозитории

## Назначение

Плагин работает как единая точка входа для инструментов обработки данных и подготовлен для дальнейшего добавления в менеджер плагинов или store-репозиторий.

## Состав поставки

- основной конфиг плагина: `config.json`
- store-метаданные: `variations[0].store`
- deploy-артефакт: `deploy/data_processing.plugin`

## Встроенные модули

- `xlsx_compare_files`
- `xlsx_analyze_files`
- `clickhouse`
- `olap`
- `macro_checker_card`
- `fileservice_card`
- `macro_exceljs_card`

## Примечания

- `r7_macro_checker` и `r7_fileservice` оставлены коммерческими карточками, потому что требуют серверную часть.
- `r7_macro_excelJS` также оформлен карточкой, чтобы не ломать текущую архитектуру каталога.
- Скриншоты для store подготовлены отдельно в `resources/store/screenshots`.
