{r7c}.code Редактирование макросов Р7

Бесплатная версия для некоммерческого использования

Инструмент для создания и редактирования JavaScript‑макросов в редакторах R7. Сфокусирован на удобной разработке, отладке и интеграции с данными.

  • Интерфейс в стиле VS Code: вкладки, хлебные крошки, сплит‑экран и быстрые переключения между файлами.
  • AI Sputnik: подключение провайдеров, быстрые подсказки и чат для ускорения разработки (правая панель).
  • Доступ к внешним файлам : встроенный SQLite/WASM‑движок для чтения CSV, Excel и других табличных форматов с пагинацией результатов.
  • Внешние скрипты: подключайте JS/VBA‑файлы как библиотеку решений; дерево группируется по файлам.
  • Импорт Excel VBA: импорт проектов и настраиваемая трансляция кода в JavaScript.
  • Синтаксический анализ кода: подсветка ошибок и предупреждений напрямую в редакторе.

  • Журнал логов: фильтрация, копирование и экспорт сообщений для удобной отладки.
  • Debug Insights: отчёты «Покрытие кода», «Использование API», «Линтер», «Производительность» и «Граф зависимостей». Откройте правую панель и нажмите кнопку с иконкой графа. Отчёты обновляются при каждом открытии.
  • 700+ примеров API R7: готовые фрагменты на вкладке «Примеры» для быстрого старта.

  • Сравнение скриптов: два редактора бок о бок для визуального сравнения и правок.
  • Быстрый обзор документа: сворачивание панелей для просмотра данных без закрытия редактора.
