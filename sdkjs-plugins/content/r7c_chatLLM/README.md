# {r7c}.ChatLLM

`{r7c}.ChatLLM` is an AI copilot built right into R7/ONLYOFFICE. It helps users write faster, rewrite text more clearly, work through spreadsheet tasks, and generate ideas without leaving the editor.

## What it helps with

- answering questions about the current document or task;
- rewriting text in a clearer, shorter, or more persuasive way;
- drafting emails, instructions, summaries, and presentations;
- helping with spreadsheet wording, formulas, and table-related tasks;
- generating ideas, options, headlines, and first drafts.

## Why users like it

- everything happens inside the editor, in context;
- there is no need to switch between multiple services and tabs;
- it works well for documents, spreadsheets, and presentations;
- it turns blank-page moments into a faster start.

## Who it is for

Useful for managers, analysts, marketers, educators, support teams, sales teams, and anyone who wants to move from rough idea to polished result faster.
