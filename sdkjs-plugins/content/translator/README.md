## Overview 

Translate the selected text into other languages.

The plugin uses Google Translate API. All the languages supported by the service are supported by the R7-Office plugin.

It is installed by default in cloud, [self-hosted](https://github.com/R7-Office/DocumentServer) and [desktop version](https://github.com/R7-Office/DesktopEditors) of R7-Office editors. 

## How to use

1. Select text you want to translate.
2. Open the Plugins tab and press Translator.
3. Select the language you wan’t your text to be translated to.

If you need more information about how to use or write your own plugin, please see this https://api.R7-Office.com/docs/plugin-and-macros/get-started/overview/

## Known issues

* Doesn't work on Internet Explorer (bug #47773 in the internal bug tracker)
* Doesn't work in China.
* Doesn't work on XP system.
