# Game Wizard

`Game Wizard` — это небольшая коллекция мини-игр прямо внутри редактора. Когда нужен короткий перерыв, не надо уходить в браузер и выпадать из рабочего процесса.

## Что получает пользователь

- знакомые игры, собранные в одном лаунчере;
- быстрый запуск из одного окна;
- удобный способ переключиться на пару минут и вернуться к задаче;
- аккуратный интерфейс, который ощущается частью рабочего пространства.

## Что внутри

- 2048
- Косынка
- Паук
- FreeCell
- Шахматы

## Почему это удобно

Иногда достаточно нескольких минут передышки, чтобы вернуться к работе с новой концентрацией. `Game Wizard` даёт такой перерыв рядом, без лишних вкладок и отвлечений.
