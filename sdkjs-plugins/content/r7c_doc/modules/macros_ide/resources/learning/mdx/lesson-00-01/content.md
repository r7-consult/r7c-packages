# Введение

«Аналитические базы данных» (OLAP) в плагине «Слайдер Данные» предназначен для многомерного анализа информации, который позволяет рассматривать данные с разных точек зрения (измерений) и на различных уровнях детализации. В отличие от работы с плоскими таблицами (SQL), OLAP оперирует такими понятиями, как кубы (многомерные структуры данных), измерения (категории для анализа, например, время, товар, регион) и меры (числовые показатели, например, продажи, прибыль). Этот модуль является импортозамещающим решением для таких инструментов, как Microsoft Analysis Services и Power BI Pivot Tables, и позволяет создавать интерактивные сводные отчеты (pivot-таблицы) напрямую в табличном редакторе Р7-Офис. Вы сможете подключаться к готовым OLAP-кубам, а также создавать свои собственные аналитические модели (ROLAP) на основе данных из обычных баз данных, таких как PostgreSQL. Это открывает возможности для глубокого финансового, маркетингового и управленческого анализа без необходимости написания сложных SQL-запросов.
