# Hello World

`Hello World` — это простой демонстрационный плагин, который показывает, как расширение может выглядеть в редакторе и как оно взаимодействует с выделенным текстом.

## Зачем он нужен

- чтобы быстро увидеть базовый сценарий работы плагина;
- чтобы понять, как пользователь воспринимает окно расширения;
- чтобы взять готовую основу для своего первого простого решения.

## Что показывает

- открытие отдельного окна плагина;
- вывод приветственного сообщения;
- работу с выделенным текстом из документа.

## Для кого

Подойдёт тем, кто хочет познакомиться с расширениями R7/ONLYOFFICE на простом и понятном примере.
