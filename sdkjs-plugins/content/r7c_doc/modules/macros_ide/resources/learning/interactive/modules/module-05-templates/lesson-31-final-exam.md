# Урок 31. Финальный Контроль: Гранд-Экзамен Архитектора Р7

Поздравляем! Вы прошли долгий путь от простейших скриптов `.GetValue()` до понимания `Strict Mode`, `Try/Catch` инкапсуляции и глубокой объектной модели Р7-Офис `(V8 DOM Architecture)`.
Вы больше не переписываете VBA построчно. Вы проектируете надежные плагины "в браузере". Вы понимаете, что такое Прямой Цикл `Delete` (и почему он ломает индексы), вы знаете как прятать листы от бухгалтеров, и зачем писать `RecalculateAllFormulas()` при смене данных.

Венец вашего обучения — это **Гранд-Экзамен**. Вам предстоит самостоятельно, с чистого листа внутри защитной капсулы `try/catch` реализовать сведение всех модулей.

## 🏁 Архитектурное задание (Senior Level)

Справа вас ждет каркас Эталонного Макроса (с проверкой `Api` и перехватом ошибок).
Ваша цель — дописать ядро, состоящее из 4 логических слоев:

1. **Фабрика (М4 / М5):** 
   - Сгенерируйте через API новый рабочий лист `ApiWorksheet` с именем `"Project-VIP"`. 
   - *Мы уже написали для вас команды, чтобы вы не отвлекались на имена переменных.*

2. **UI Вёрстка (М2):** 
   - На созданном листе `"Project-VIP"` инстанциируйте блок ячеек `"A1:D1"`.
   - Выполните глобальное блочное слияние ячеек с помощью метода `.Merge(false)`.
   - Заложите туда строковое значение `"СЕКРЕТНЫЙ ПРОЕКТ"`.
   - Центрируйте его по оси X (Горизонталь) методом `.SetAlignHorizontal("center")`.
   - Модифицируйте `Background Color` ячейки (Заливка) на Красный цвет `RGB(255, 0, 0)`. Не забывайте использовать `Api.CreateColorFromRGB()` (Мы отработаем Урок 6).

3. **Интерфейс Сообщений (М5):**
   - Используйте глобальный метод Р7-Офис `.AddComment()` (как в шаблонах разработчиков Р7-Consult), чтобы добавить документный комментарий: Текст `"Экзамен пройден"`, Автор `"Экзаменатор"`. (Не привязывайте его к ячейке или Листу, метод вызывается глобально `Api...`).

4. **Математика и Синхронизация Кэша (М3 / М5):**
   - Вернитесь к листу `"Project-VIP"`. Внесите хардкод (Константу) `50` в ячейку `"A2"`.
   - В ячейку `"C2"` внесите формульную строку: `"=A2*100"`.
   - Из-за асинхронности браузерного движка таблица может счесть это "отложенным обновлением". Как Профессионал, вы должны принудительно пнуть калькулятор книги: **вызовите команду пересчета всех формул `Api.RecalculateAllFormulas()`**.
