Этот пакет включает или переиспользует сторонние компоненты из upstream-репозиториев.

Из известных зависимостей и вендорных частей:

- runtime-ассеты плагинов ONLYOFFICE / R7 в `vendor/oo`
- SheetJS `xlsx.full.min.js`
- jQuery и jQuery UI внутри импортированных модулей
- FancyTree внутри импортированных модулей
- jspreadsheet / jsuites внутри импортированных модулей
- Font Awesome внутри импортированных модулей

Каждый сторонний компонент продолжает использоваться на условиях своей исходной лицензии.

При переносе плагина в store-репозиторий нужно сохранять исходные лицензионные уведомления рядом с перераспределяемыми файлами.
