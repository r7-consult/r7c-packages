# Расчет налогов по условиям

Вам предстоит автоматизировать расчет налога для списка контрагентов. Ставка налога зависит от нескольких критериев (доход и тип юридического/физического лица).

## Условия расчета

* **"Физ. лицо"**: базовая ставка `13%` (0.13).
  * Если доход Физ. лица превышает `100 000`, ставка возрастает на `2%` (итого `15%`).
* **"Юр. лицо"**: базовая ставка `20%` (0.20). Дополнительных прогрессивных ставок нет.
* Во всех остальных случаях (пустой доход, неизвестный тип лица), налог равен `0`.

Доходы указаны в столбце `A`, тип лица в столбце `B`. Результаты расчета необходимо записать в столбец `C` для каждой строки в диапазоне (со строки 2 по строку 8).

Желательно использовать чтение и запись массивом (`GetValue()` и `SetValue()` для нужного Range) для повышения производительности скрипта.
