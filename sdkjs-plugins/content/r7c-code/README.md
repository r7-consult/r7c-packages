# {r7c}.code

`{r7c}.code` помогает быстрее автоматизировать работу с документами и таблицами в R7/ONLYOFFICE. Это удобное место, где рутинные действия превращаются в понятные сценарии, а повторяющиеся операции занимают секунды вместо ручной возни.

## Что можно делать

- собирать полезные макросы под свои повседневные задачи;
- ускорять типовые операции с документами и таблицами;
- сокращать количество ручных действий и случайных ошибок;
- поддерживать единый стандарт работы внутри команды.

## Чем особенно ценен

Плагин подходит тем, кто хочет не просто «что-то автоматизировать», а реально освободить время для основной работы. Всё нужное находится рядом с документом, поэтому сценарии проще запускать, проверять и использовать каждый день.

## Для кого

`{r7c}.code` будет полезен специалистам, которые регулярно готовят документы, работают с шаблонами, таблицами и повторяющимися процессами.
