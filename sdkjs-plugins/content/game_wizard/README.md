# Game Wizard

`Game Wizard` brings a compact game shelf into the editor, so users can take a short break without leaving their workspace.

## What you get

- familiar mini games collected in one launcher;
- quick start from a single window;
- an easy way to reset your focus during a busy day;
- a lightweight experience that feels native inside the editor.

## Included games

- 2048
- Klondike Solitaire
- Spider Solitaire
- FreeCell
- Chess

## Why it is useful

Sometimes a five-minute pause is enough to return with a clear head. `Game Wizard` keeps that pause close and convenient, without sending the user into a browser rabbit hole.
