# Change Log

## v0.1.0

- Добавлен плагин для записи действий пользователя в R7 Spreadsheet и воспроизведения последнего записанного сценария.
- Добавлена генерация JS-макроса и `replayPlan` для воспроизведения записанных действий.
- Добавлена запись значений, чисел, формул, форматирования ячеек, границ, `merge/unmerge` и финального выделения.
- Добавлено хранение состояния в скрытом листе книги `.r7c_macro_recorder_store` и резервный backup в `localStorage`.
- Добавлен компактный режим окна во время записи с отдельной кнопкой `Стоп` и отображение записанного кода в режиме `read-only`.
- Интерфейс плагина ориентирован на сценарий `Запись -> Стоп -> Запустить` без ручного редактирования кода.
