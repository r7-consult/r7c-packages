# Очистка данных

В реальных задачах данные часто приходят в "грязном" виде: с лишними пробелами, в разном регистре. Ваша задача — написать макрос, который приведет имена в колонке А к стандартному виду.

## Условия
На листе уже есть таблица с данными. В диапазоне `A2:A6` находятся имена с лишними пробелами по краям и неправильным регистром букв.

**Вам нужно:**
1. Прочитать данные из диапазона `A2:A6`.
2. Убрать пробелы в начале и в конце каждой строки.
3. Сделать первую букву заглавной, а все остальные — строчными.
4. Записать очищенные данные обратно в тот же диапазон `A2:A6`.

**Пример:**
`"  иВанов иван "` ➔ `"Иванов иван"`
`"ПЕТРОВ ПЕТР"` ➔ `"Петров петр"`

*Удачи!*
